# Next.js & Fidely UI Template

A starter template for building modern web applications with **Next.js (Page Router)** and **Fidely UI** 

## Technologies Used

- Next.js 16 (Page Router)
- Fidely UI - latest
- Panda CSS
- TypeScript
- next-themes

## Getting Started

### Create a new project

Create a new project using this template with `create-next-app`:

```bash
npx create-next-app -e https://github.com/fidely-ui/next-page-template
```

### Install dependencies

### Run the development server

Start the development server:

```bash
npm run dev
```
